# CLIENT_CONTACTS_APP

## Запуск проекта
1. Запустить серверную часть приложение (https://github.com/Codiscodi/Server_Contacts_App.git)
2. Склонировать Клиентскую часть приложения
3. Установить модули
4. Запустить приложение, прописав в командной строке `npm start`
